# 시각-전정 감각 불일치 환경에서의 사이버멀미 연구 (Cybersickness in Sensory Conflict Environments)

본 프로젝트는 몰입형 미디어 환경에서 발생하는 사이버멀미(Cybersickness)의 근본 원인인 '시각-전정 감각 불일치'를 인간공학적 관점에서 분석하고, 이를 설계 단계에서 완화할 수 있는 **방향성 주변부 블러(Directional Peripheral Blur, DPB)** 기법을 제안합니다[cite: 1].

## 📌 연구 개요
본 연구는 사용자 개인의 환경 조절(자세, 휴식 등)만으로는 사이버멀미를 근본적으로 해결하기 어렵다는 점에 착안하여, 설계 단계에서 시각적 자극을 선택적으로 약화시키는 방법론을 검증하였습니다.

*   **교과목:** IMEN315 인간공학 (고려대학교 산업경영공학부)
*   **지도교수:** 오형석 교수님
*   **팀원:** 곽준우, 김상평, 천세환, 박찬우, 서수인

## 🔍 핵심 연구 내용
1.  **문제 정의:** 로드뷰 및 몰입형 미디어에서 발생하는 감각 불일치 메커니즘 분석
2.  **사용자 관점 검토:** 개인의 물리적 환경 조절(거리, 조도, 자세 등)이 멀미 완화에 미치는 영향 및 한계 분석
3.  **설계자 관점 실험(DPB 제안):**
    *   **방향성(Directional):** 이동 방향 반대편(Trailing edge) 주변부에만 블러 처리하여 시각적 불일치 감소
    *   **최소 개입(Minimal Intervention):** JND(지각 임계치) 이하의 블러 강도를 사용하여 사용자 인지 최소화
    *   **중앙 보호(Center Protection):** 정보가 집중되는 화면 중앙부는 블러 처리 제외

## 📊 주요 실험 결과
본 실험 결과, 방향성을 적용한 DPB-JND 이하 조건이 기존의 무처리 조건 및 전방위 블러 조건보다 사이버멀미를 유의미하게 감소시켰습니다.
*   DPB는 시각적 방해를 절반 수준으로 줄이면서도 효과적으로 멀미를 완화하였습니다.
*   강한 개입보다는 최소 개입 기반의 방향성 설계가 사용자 경험(몰입감)과 멀미 저감 사이의 최적 균형을 보여주었습니다.
